# Placeholder Images

## placeholder-car.svg
This is an SVG placeholder that displays when car images fail to load or are not available.

## To use a JPG placeholder instead:
1. Add a `placeholder-car.jpg` file to this directory
2. The CarImage component will automatically use it as a fallback

## Recommended Dimensions:
- Width: 800px
- Height: 600px
- Aspect Ratio: 4:3
- Format: JPG, PNG, or SVG
